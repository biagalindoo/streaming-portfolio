# Streaming Portfolio

Projeto de portfólio de streaming de séries.

## Backend
- Node.js
- Autenticação de usuários
- Catálogo de séries

## Como rodar
1. Entrar na pasta backend:
cd backend
2. Instalar dependências:
npm install
3. Rodar o servidor:
node src/server.js
4. Rotas:
- `GET /catalog` → lista de séries
- `POST /auth/login` → login de usuário (JSON)
